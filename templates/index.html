Can Opener

Welcome to Can Opener, a small application that explores the (controversial) license plate tracking data in Minneapolis, MN.

This application's main goal is to provide context to a controversial dataset and to aid in intelligent conversation about this and other datasets. It is not meant to be commentary on the value or privacy concerns of the data.

The data was legally obtained from the Minneapolis Police Department on Dec 11, 2012. The data contains records from Aug 30, 2012 through Nov 29, 2012.

There is currently state legislation being proposed to reform the restrictions on this data.

Though the dataset is from a slightly different time, Datalytics Science did an excellent in-depth analysis of this data.

Feel free to contact me if you have any questions or comments. Also note that this application has costs for hosting so it will not remain up indefinitely.
